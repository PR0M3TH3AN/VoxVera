### Version: v0.1.7

#### 🚀 New Features
- **Official Uninstallers:** Added cross-platform uninstallation scripts (`uninstall.sh` and `uninstall.ps1`) to safely remove the VoxVera CLI tool and its dependencies from user systems without affecting generated flyers.
- **Enhanced Documentation:** Usage guides across all 14+ languages have been updated to include complete instructions for the new uninstallation process.

#### 🛠 Improvements
- **Simplified Viral Distribution:** The download button on all generated flyers has been streamlined. Instead of a messy dropdown with multiple installers, users now receive a single, clean `voxvera-source.zip`. 
- **Offline-Ready Source Bundles:** The source ZIPs generated by the flyers now fully encapsulate the `vendor/` dependency directory, guaranteeing that anyone who downloads the tool from a flyer can run it offline instantly.
- **Release Automation Upgrades:** Hardened the automated release process (`RELEASE.md`) to explicitly mandate that all documentation templates are translated across all supported languages before any new release is cut.

#### 🐛 Bug Fixes
- **Test Suite Rigidity:** Hardened the automated integrity testing suite to physically verify the presence of uninstall scripts and embedded modules, rather than relying on stubbed dummy data. All 31 tests are passing with 100% green integrity.
